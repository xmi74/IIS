Implementace

Na MySql databázu sa ovládame pomocou ORM SqlAlchemy. Python skripty v zložke models reprezentujú tabulky v databáze. Metódy v zložke api implementujú CRUD prácu s ORM triedami. Zložka forms obsahuje python skripty, ktoré implementujú formuláre v aplikácií. custom_validators.py implementuje naše vlastné špecifické validátory na formluláre. routes.py implementuje cesty k jednotlivým stránkam a spracuváva HTTP požiadavky na tieto cesty. templates obsahuje dynamické HTML stránky, kde dynamickosť zabezpečuje framwork Jinja. app.py inicializuje celú aplikáciu.
